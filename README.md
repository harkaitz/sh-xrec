XREC
====

## Help

xrec

    Usage: xrec { -V | -l | -r JOB | -p JOB | -c JOB} 
    
    Record screen when doing a job, this way you can remember where
    you where at when connecting.

xrec-systray

    Usage: xrec-systray -V
    
    Control xrec(1) from the systray using sicon(1).
